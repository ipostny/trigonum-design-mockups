ТЗ — вкладка «Риск-менеджмент» (роль: Риск-менеджер)
1) Где находится и общая структура
Раздел «Риск-менеджмент» в верхней навигации.
Внутри раздела два таба:
Команда — основная таблица по закреплённым за риск-менеджером трейдерам.
Политики — текущий список политик (как есть), добавляется просмотр «кто следует политике».
2) Таб «Команда»
Таблица со списком трейдеров под управлением данного риск-менеджера.
Колонки (слева → вправо):
Трейдер — имя/ник.
Политика РМ — дропдаун, список из реестра «Политики риска»; выбор мгновенно назначает политику трейдеру. (Поведение идентично выбору политики в карточке трейдера. Рынок в политике — не обязателен.) 
Остаток риска — прогресс-бар (0–100%) с подсказкой «остаток/лимит».
Предупреждения — число; кликабельно → поп-ап «Нарушения»: список сработок с датой/временем, типом нарушения и ссылкой «Открыть сделки». По факту общая таблица сделок с фильтром по трейдеру и по статусу закрытия
Принудительно закрытые — число; кликабельно → поп-ап «Сделки по принудительному закрытию» (таблица сделок). По факту общая таблица сделок с фильтром по трейдеру и по статусу закрытия
Действия — кнопки:
Назначить беседу — поп-ап с датой/заметкой; сохраняет событие в системе и помечает трейдера «ожидает беседу».
Назначить тестирование — ставит флаг обязательное тестирование; торговля трейдера блокируется
Запретить торговлю (тумблер) — мгновенно блокирует/разблокирует торговлю.
Запрет по монете — мультиселект монет → сохранить (трейдеру запрещено открывать сделки по выбранным монетам).
Взаимодействия и поведение:
Клик по предупреждению/нарушению → список сделок, где нарушение сработало, формат как во вкладке «Сделки», плюс колонка «Причина (правило РМ)», почему сделка была не допущена/закрыта.
Сортировка по всем колонкам; фильтры по статусам (есть нарушения, тест назначен, торговля запрещена и т. п.).
Пагинация стандартная. Пустые состояния: «Нет данных», «Нарушений нет».
3) Таб «Политики»
Отображается существующий список политик без изменений логики.

Таблица должна быть как сверху, скрин для примера прогресс бара – остатки рисков
