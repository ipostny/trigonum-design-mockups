ТЗ — вкладка «Дневник трейдера»
1) Где находится
В верхней панели у страницы трейдера только три вкладки: Обзор | Торговля | Дневник трейдера.
«Дневник трейдера» — отдельная вкладка (не часть «Обзора» и не «Торговли»).
2) Состав вкладки
Место для выгрузки торговой стратегии
История сделок
Подвкладка 1: «Торговая система»
Представление:
Текст: встроенный простой редактор (заголовки, списки, ссылки).
Файл (опционально): карточка загруженного DOC/DOCX/PDF с кнопкой Скачать.
Если есть и текст, и файл — сверху текст, ниже карточка файла.
Под заголовком показать строку: «Последнее обновление: ДД.ММ.ГГ ЧЧ:ММ, Автор».
Кнопки/действия:
Для трейдера: Редактировать текст, Загрузить/Заменить файл, Сохранить, Отмена.
Для Тим-лида / Старшего трейдера / РМ: только просмотр и Скачать (отображается в карточке аналитики трейдера)
Подвкладка 2: «История сделок»
Назначение
Список сделок трейдера (открытые и закрытые) с возможностью добавить/изменить информацию в поп-апах для конкретной сделки.
Фильтры над таблицей
Период, Инструмент, Направление (LONG/SHORT), Статус (Открыта/Закрыта), Плечо (диапазон).
Таблица (слева → вправо)
Дата/время
Инструмент
Направление
Объём
Цена входа
Текущая/Цена выхода (в зависимости от статуса)
P&L, $
P&L, %
Тейк-профит (число)
Стоп-лосс (число)
Причины входа (заполнены/нет)
Взаимодействие
Клик по строке или по ячейкам «Причины» открывает поп-ап «Детали сделки».
В поп-апе трейдер может редактировать: Причину входа, Комментарий, SL, TP, RR, Риск на сделку .
В P&L — подсветка: прибыль зелёным, убыток красным.
Права
Трейдер — добавляет/редактирует данные в поп-апе своих сделок.
Тим-лид / старший трейдер / риск-менеджер — просмотр (без редактирования).
Пустые состояния
«Нет сделок за выбранный период».
