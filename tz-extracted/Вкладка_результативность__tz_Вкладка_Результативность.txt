ТЗ — вкладка «Результативность» (инвестор)
1) Где находится
Нижний таббар: Главная · Инвестирование · Отчётность (активна) · Баланс.
Название экрана: «Отчётность».
2) Фильтры (в верхней полосе, одна строка)
Период: Месяц / Квартал / Год.
Минимальная единица отчётности — МЕСЯЦ. Никаких дневных разбиений.
Для Года показываем только помесячно (12 точек/столбиков).
Стратегия: дропдаун из стратегий, куда вложено (+ пункт «Все стратегии» по умолчанию).
Справа от фильтров — кнопка «Скачать PDF».
3) Верхние показатели (карточки)
Суммарный P&L за период (в валюте счёта, и %).
Капитал на начало периода.
Капитал на конец периода и Δ за период (в $ и %).
(Только эти цифры; без рейтингов и «за день».)
4) Графики (сразу под карточками)
Суммарный P&L
Представление: столбцы по месяцам (для любого выбранного периода).
Подписи: «Месяц», «P&L».
Тултип по наведению: «Месяц, P&L $ / %».
P&L по стратегиям
Представление: группированные столбики (одна группа = месяц; внутри — столбики по стратегиям, выбранным в фильтре, либо топ-N + «прочее», если их много).
Легенда с цветами стратегий.
Тултип: Стратегия, Месяц, P&L.
Для Месяца/Квартала ось X всё равно помесячная: текущий месяц отображается одной колонкой (month-to-date).
5) Нижние табы (переключатели внутри экрана)
Три вкладки под графиками:
Доходность (мес.) — линейный график доходности по месяцам (кумулятивная линия или помесячная, выбрать один вариант; активный месяц подсвечен).
Капитал — линия equity по месяцам (начало/конец каждого месяца; точка месяца = капитал на конец).
Распределение — две карточки рядом/друг за другом:
P&L по монетам — круговая диаграмма (доли по монетам за выбранный период).
Соотношение Long/Short — пончик/полоска 100% (доли по направлению за период).
6) Действия и взаимодействия
Любая колонка/точка на графиках кликабельна → перерисовка всех виджетов по выбранному месяцу (в рамках периода).
Кнопка «Скачать PDF» выгружает: выбранные фильтры, карточки, оба верхних графика и активную нижнюю вкладку.
Нет «рейтинга по P&L».
Нет «суммарного P&L за день».
Все подписи и цвета: прибыль — зелёный, убыток — красный.
