Getting Started

Last updated: March 2026

Welcome to TrigonumTrade

TrigonumTrade is a professional crypto trading platform designed for teams. This guide will help you get started with the key features and workflows.

Quick Setup

  1. Connect your Bybit exchange account via API keys
  2. Configure your trading preferences and risk limits
  3. Set up your team members with appropriate roles
  4. Start trading from the Terminal page

Key Features

The platform provides several core capabilities:

  • Trading Terminal — Real-time charting with TradingView and order management
  • Risk Management — Daily limits, position controls, and policy enforcement
  • Reports — Comprehensive P&L analysis, performance tracking, and exports
  • Investor Portal — Dedicated interface for investment program management

System Requirements

TrigonumTrade works best with modern browsers:

  • Chrome 100+ (recommended)
  • Firefox 100+
  • Safari 16+
  • Edge 100+
